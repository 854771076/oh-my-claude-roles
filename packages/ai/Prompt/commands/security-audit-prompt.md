# 提示词安全合规审计

执行企业级提示词安全合规检查，覆盖所有强制安全防护要求。

## 执行步骤
1. **输入层防护检查**
   - 确认所有用户输入经过Zod Schema校验，过滤非法字符与超长内容
   - 确认已接入Lakera Guard检测注入攻击、越狱尝试与敏感信息，检测到攻击直接拦截
   - 确认实现角色隔离，用户输入无法覆盖系统提示词规则
2. **模型层防护检查**
   - 确认系统提示词加入角色锁定规则，禁止模型泄露/复述/修改系统提示词
   - 确认明确能力边界，禁止执行未授权操作
   - 工具调用场景确认：工具权限范围受限，执行前经过二次权限校验
3. **输出层防护检查**
   - 确认所有输出经过Zod Schema校验，不符合格式直接拦截
   - 确认输出内容经过合规审核与敏感信息过滤
   - 高风险场景确认要求人工审核后再输出
4. **合规管控检查**
   - 确认符合属地监管政策要求，无违规生成可能
   - 专项场景检查：金融/医疗/法律等场景包含强制免责声明，无违规服务风险
   - 确认无个人信息违规处理，无知识产权侵权风险
   - 确认无违规伦理内容生成可能

## 检查要点
- ✅ 三层防护完整，无安全漏洞
- ✅ 注入越狱拦截规则到位，无绕过可能
- ✅ 所有合规要求覆盖，符合监管政策
- ✅ 敏感信息保护到位，无泄露风险
